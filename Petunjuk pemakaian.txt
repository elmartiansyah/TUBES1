Petunjuk instalasi Wapkul :

- Upload semua file ke web server/hosting di /public_html
- Buka alamat : http://situs-anda.com/install.php
- kemudian ikuti petunjuk instalasi.
- Untuk masuk Admin panel : http://situs-anda.com/admin-panel
- Untuk masuk Cpanel Shoutbox gunakan password default 12345, silahkan ubah demi keamanan lewat ftp dengan mengedit shoutbox/shout.php
